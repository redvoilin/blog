== README

此项目是用ruby on rails编写的，自动化测试框架的样例test case可以对此项目搭建的web site进行测试

如何安装和启动此项目:

1.安装ruby
  windows可以通过这个地址安装：http://railsinstaller.org/en

  osx和linux推荐通过rvm来安装和管理ruby：

  1.安装rvm：https://www.rvm.io/

  2.通过rvm安装ruby: rvm install 2.2.1

2.安装所需要的gem
   
  1.进入blog目录
   
  2.运行此命令安装所需gem: <tt>bundle install</tt>
